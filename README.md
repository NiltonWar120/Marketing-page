# Agencia de Marketing – Nilton Guerra 🧑‍💻✨

Sitio web para una agencia de marketing.

## 📋 Descripción

landing page moderno para una agencia de marketing desarrollado con React y Vite que presenta los servicios que ofrece la agencia NEXUS . Incluye secciones de servicios, benefios,about, procesos, testimonios, trabajo y contacto, con animaciones profesionales y una interfaz clara enfocada en clientes.

## ✨ Características

- **Diseño responsivo**: Interfaz adaptable a móviles, tablets y desktop
- **Animaciones con GSAP**: Entradas suaves, efectos de scroll y microinteracciones
- **Efectos visuales**: Fondos con parallax, avatar 3D con glitch y detalles hover en tarjetas

## 🗂️ Estructura del Proyecto

```
├── public/
│   ├── favicon.svg                # Svg que representa el logo de la empresa
├── src/
│   ├── components/             # Cursor, Footer, HeroCanvas, Navbar, ...
│   ├── Sections/               # Hero, About, Benefits, Process, Services,Testimonials, ...
│   ├── Stiles/                 # Componente dee stilos   
│   └── main.jsx                # Punto de entrada de React
└── index.html                  # Plantilla base
```

## 💻 Tecnologías Utilizadas

- React 18
- Vite
- JavaScript (ES6+)
- CSS3 (variables, flexbox, grid, keyframes)
- three.js (estilos 3d)
- particle.js (interaccion con particulas dentro de la pagina)
- GSAP (animaciones y timelines)

## 🚀 Cómo Usar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/NiltonWar120/Marketing-page.git
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Ejecuta en desarrollo:
   ```bash
   npm run dev
   ```

4. Abre el enlace del servidor en tu navegador y navega por las secciones.

## 📦 Scripts Disponibles

- `npm run dev` - Inicia el servidor de desarrollo
- `npm run build` - Genera la versión de producción
- `npm run preview` - Previsualiza la build de producción
